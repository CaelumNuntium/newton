Команда make собирает исполнимый файл newton. 
После сборки при запуске программы командой ./newton решается уравнение f(x) = 0.
	Функция f(x) - векторная или скалярная функция, описанная в файле f.h
Если f - векторная функция, то результат записывается в файл result.dat; если f - скалярная, то результат 
записывается в файл result.dat и выводится на экран.
